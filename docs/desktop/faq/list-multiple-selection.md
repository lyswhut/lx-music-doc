title: 列表多选
---

从 v0.18.0 起，列表多选需要键盘配合。

按住 <kbd>Shift</kbd> 或 <kbd>Ctrl</kbd> 键，再通过鼠标点击想要选中的内容，即可触发多选机制。

其中 <kbd>Shift</kbd> 键用于连续选择，<kbd>Ctrl</kbd> 键用于不连续选择。

当列表获取焦点时，按 <kbd>Ctrl</kbd> + <kbd>A</kbd> 可以快速全选。

*若按了没效果，则需要点一下列表区域，使列表获取焦点。*

- 例一：想要选中 1~5 项。先按住 <kbd>Shift</kbd> 键，随后鼠标点击第 1 项，再点击第 5 项即可完成选择；
- 例二：想要选中第 1 项与第 3 项。先按住 <kbd>Ctrl</kbd> 键，随后鼠标点击第 1 项，再点击第 3 项即可完成选择；
- 例三：想要选中当前列表的全部内容。先按住 <kbd>Ctrl</kbd> 键不放，然后按 <kbd>A</kbd> 键，即可完成选择。

配合 <kbd>Shift</kbd> 或 <kbd>Ctrl</kbd> 选择时，点击未选中的内容会将其选中，点击已选择的内容则会其取消选中。

若想全部取消选择，只需放开 <kbd>Shift</kbd> 或 <kbd>Ctrl</kbd> 键，随意点击列表内的一项内容即可。


:::tip
在 macOS 上 <kbd>Ctrl</kbd> 键对应 <kbd>Command</kbd> 键。
:::

:::tip
选好内容后，可鼠标右击弹出右键菜单，对已选内容进行对应操作。
:::
