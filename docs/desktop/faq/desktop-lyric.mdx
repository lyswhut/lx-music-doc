### Windows 7 桌面歌词显示异常

Windows 7 未开启 Aero 效果时，桌面歌词显示会出现问题。详情请参阅常见问题「[界面异常 → Windows 7 界面异常](./interface-abnormal#windows-7-界面异常)」章节。

### macOS 桌面歌词有残留阴影

v2.8.0 及更高版本已修复该问题。

### Linux 桌面歌词窗口异常

v1.2.1 以前的版本，在 Ubuntu 18.10 系统上第一次开启桌面歌词时，歌词窗口可能会变白，此时需要关闭后再重新开启。

v1.2.1 及更高版本已修复该问题。

其他 Linux 系统未测试，如有异常也属意料之中。目前暂不打算处理 Linux 平台的桌面歌词兼容性问题，但你可以参考常见问题「[界面异常 → Linux 界面异常](./interface-abnormal#linux-界面异常)」提供的方案尝试解决。
