title: 可用的鼠标、键盘快捷操作
---

- 鼠标右击播放栏的歌曲专辑封面，可以定位当前播放的歌曲。
- 鼠标右击播放栏进度条上的 `LRC` 按钮，可以锁定/解锁桌面歌词。
- 歌曲搜索框、歌单链接输入框内右击，可以将剪贴板内容粘贴到输入框。
- 鼠标右击搜索界面中的单条搜索历史，可将其移除。
- 选中歌曲列表内的文字后右击，可以复制选中的文字。
  - 此功能只对「搜索」「歌单」「排行榜」「我的列表」有效。
- 鼠标在播放详情页内右键双击，可以关闭播放详情页。
- 鼠标左击播放栏上的歌曲名称，可以将其复制。
- 鼠标右击「设置 → 基本设置 → 更多主题」中的「道法自然」主题，可以打开亮、暗主题设置窗口。
- 歌曲搜索框的候选内容可用键盘上下方向键选择，按回车键搜索已选内容。
- 在歌单详情页按 <kbd>Backspace</kbd>，可返回歌单列表。
- 歌曲列表中可以使用 <kbd>Ctrl</kbd>/<kbd>Shift</kbd> 键进行多选，操作类似于 Windows 资源管理器的文件选择。
  - 详情请参阅常见问题「列表多选」章节。
- 在「我的列表」内可使用 <kbd>Ctrl</kbd>+<kbd>F</kbd> 键打开列表搜索框，按 <kbd>Esc</kbd> 键键关闭。
  - 搜索框内按上下方向键可以选择歌曲，按 <kbd>Enter</kbd> 键跳转到该歌曲，按 <kbd>Ctrl</kbd>+<kbd>Enter</kbd> 可跳转并播放。
- 在「我的列表」按住 <kbd>Ctrl</kbd> 键可进入「拖动模式」。此时通过鼠标拖拽即可调整列表顺序。
- 按 <kbd>Esc</kbd> 键可以退出全屏状态。
- 在编辑列表名时，按 <kbd>Esc</kbd> 键可以取消编辑。
- 按 <kbd>F11</kbd> 可切换全屏状态。
- 在「添加歌曲」对话框中，若按住 <kbd>Ctrl</kbd> 键点击列表名，对话框将不会自动关闭。
  - 当需要将同一首（一批）歌曲添加到多个列表时，这会很有用。
- 鼠标悬停在播放栏的音量按钮上并滑动滚轮，可以快速调整音量。

:::tip
在 macOS 上 <kbd>Ctrl</kbd> 键对应 <kbd>Command</kbd> 键。
:::
