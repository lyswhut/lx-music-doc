title: 息屏自动暂停/无法自动播放下一曲
---

如果遇到以下问题：

- 音乐播放时突然中断。
- 在没有禁用歌曲切换的情况下，当前音乐播放完毕后没有切换到下一曲。

这可能是系统相关机制进行了限制，或者是系统本身的 Bug 导致

如果你遇到了此类问题，请排查下列内容：

- **允许自启动**：在系统权限管理中允许 LX Music 自启动。
- **忽略电池优化**：将 LX Music 设置为忽略电池优化。
  - 通常如果没有设置此项，应用会自动弹出提示框，点击「去设置」，选择「允许」或「无限制」等类似选项。
- **关闭音频卸载**：在 LX Music 的「设置 → 播放设置」中取消勾选「启用音频卸载」，然后彻底重启应用。
  - 推荐在「设置 → 基本设置」勾选「显示退出应用按钮」，并使用此按钮彻底退出应用。

通常设置完以上内容后，可以解决绝大部分的播放中断问题。如果确定排查以后仍然遇到类似问题，可以在 GitHub [提交 Issue](https://github.com/lyswhut/lx-music-mobile/issues?q=is%3Aissue+) 反馈。
